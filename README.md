# x0c/homebrew-tap

Homebrew tap for [subswap](https://github.com/x0c/subswap) and [session-continue](https://github.com/x0c/session-continue).

```bash
brew install x0c/tap/subswap
brew install x0c/tap/session-continue
```

Or tap first:

```bash
brew tap x0c/tap
brew install subswap
brew install session-continue
```
